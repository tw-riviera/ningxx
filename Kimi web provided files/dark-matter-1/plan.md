# 暗物質互動網頁 — 執行計劃

## 專案概述
創建一個名為「宇宙的奧祕：暗物質」的單頁互動式網站，目標是讓對宇宙認識不多的普通人產生興趣與驚嘆。網站需具備視覺動畫、高互動性、引人入勝的內容鋪排。

## 技能選用
- **Capability**: `vibecoding-webapp-swarm` — React + TypeScript + Tailwind CSS + shadcn/ui 的互動網頁開發

## 執行階段

### Stage 1 — 設計（Design）
- 載入 `vibecoding-webapp-swarm` 技能
- 進行設計階段：撰寫 design.md，包含：
  - 視覺風格定義（暗色宇宙主題、星空配色、霓虹光效）
  - 動畫設計（粒子系統、引力透鏡、星系旋轉）
  - 互動設計（滾動觸發、滑鼠互動、可點擊元素）
  - 技術規劃（React, Three.js/Canvas, GSAP ScrollTrigger, Framer Motion）

### Stage 2 — 開發（Development）
- 根據設計文件進行開發
- 實現多個內容章節：
  1. **Hero 區域** — 宇宙星空背景 + 標題動畫
  2. **什麼是暗物質** — 互動式粒子系統
  3. **宇宙組成** — 圓餅圖動畫（68%暗能量/27%暗物質/5%普通物質）
  4. **發現歷程** — 時間軸互動
  5. **引力透鏡效應** — 視覺演示
  6. **我們為何不知道** — 哲思式收尾
- 實現所有動畫與互動效果
- 建構與部署

### Stage 3 — 部署（Deploy）
- 部署到線上環境

## 設計方向
- 深色背景 + 星空粒子效果
- 滾動驅動的敘事體驗
- 直觀的視覺比喻取代艱深術語
- 中文內容為主
